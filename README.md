# Startup Success Prediction Dashboard

This project predicts whether a startup will succeed or fail using machine learning models.

## Features
- Dataset Explorer
- Model Accuracy Leaderboard
- Startup Prediction
- Model Comparison
- Top 10 Most Promising Startups

## ML Models Used
- Logistic Regression
- Decision Tree
- Random Forest
- SVM
- Gradient Boosting

## How to Run

1. Install dependencies
pip install -r requirements.txt

2. Run the dashboard
streamlit run app.py
